### **When Resident Creates Ticket:**

```csharp
// 1. Get available technicians sorted by rating
var technicians = await _technicianSelectionService
    .GetAvailableTechniciansByCategoryAsync(categoryId);

// Result: [Usman(4.8), Ali(4.5), Hassan(3.2)]
// Sorted by highest rating FIRST

// 2. Resident selects Usman
// 3. Ticket created directly assigned:
var ticket = await _ticketService.CreateTicketAsync(
    request,
    selectedTechnician.UserId  // Usman = ID 1
);

// 4. Database:
{
  "ticketCode": "TKT-00015",
  "statusId": 2,              // "Assigned" (not "Open")
  "selectedTechnicianId": 1,  // User selected this
  "currentTechnicianId": 1,   // Currently working
  "technicianRatingGiven": null  // Not yet rated
}

// 5. Usman notified: "TKT-00015 assigned to you!"
// 6. NO ADMIN INVOLVED!
```

### **When Resident Rates Technician:**

```csharp
// 1. Resident submits: rating=5, review="Excellent work!"
// 2. System calls:
await _technicianSelectionService.RateTechnicianAsync(
    ticketId: 13,
    technicianId: 1,     // Usman
    rating: 5,
    review: "Excellent work!"
);

// 3. System calculates new average:
double totalPoints = (4.78 * 34) + 5;  // = 167.72 + 5 = 172.72
totalRatings = 35;
newAverage = 172.72 / 35 = 4.92  // Increased!

// 4. Database updated:
Usman's User record:
{
  "averageRating": 4.92,      // Updated
  "totalRatings": 35,         // Updated
  "completedTickets": 33      // Updated
}

Ticket record:
{
  "technicianRatingGiven": 5,           // Rating stored
  "userReview": "Excellent work!"       // Review stored
}

// 5. Next resident creates ticket:
// Sees updated: Usman ????? 4.92/5 (35 reviews)
// More likely to pick Usman again!
```
