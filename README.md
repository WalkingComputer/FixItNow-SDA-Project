# FixItNow - Hostel Maintenance Ticket Management System

## 📋 Project Overview
A console-based ticket management system for hostels using Clean Architecture and Design Patterns.

## 🎨 Design Patterns Implemented
- **Singleton Pattern**: MongoDbContext
- **Factory Method**: UserFactory
- **Decorator Pattern**: TicketServiceDecorator
- **Adapter Pattern**: NotificationAdapter
- **Facade Pattern**: InputValidator
- **Observer Pattern**: NotificationObserver
- **Strategy Pattern**: TicketPriorityStrategy

## 🏗️ Architecture
- Domain Layer
- Infrastructure Layer  
- Application Layer
- Presentation Layer

## 🚀 Technologies
- .NET Framework 4.7.2
- MongoDB
- C#
